# SYDNEY 여행 일정 (PWA)

친구들과 떠나는 시드니 여행 일정을 아이폰에서 앱처럼 쓸 수 있는 PWA(Progressive Web App)입니다.

## 폴더 구조

```
/
├── index.html
├── manifest.json
├── service-worker.js
├── icons/
│   ├── icon-180.png
│   ├── icon-192.png
│   ├── icon-512.png
│   └── favicon.ico
└── README.md
```

이 zip 파일을 압축 풀지 않고 그대로 배포 서비스에 업로드하면 됩니다.

---

## 1. Netlify Drop으로 배포하기 (가장 쉬움, 추천)

1. https://app.netlify.com/drop 접속
2. 이 zip 파일을 화면에 그대로 드래그 앤 드롭
3. 몇 초 후 `https://랜덤이름.netlify.app` 같은 주소가 발급됩니다
4. 이 주소를 아이폰 사파리에서 열면 끝!

> Netlify Drop은 계정 없이도 사용 가능하지만, 나중에 다시 업데이트하려면 무료 계정을 만들어서 같은 사이트에 새 zip을 드래그하면 됩니다 (주소가 유지됩니다).

---

## 2. GitHub Pages로 배포하기

1. GitHub에서 새 저장소(repository)를 만듭니다 (예: `sydney-trip`)
2. 이 zip 파일의 압축을 풀고, 안의 파일들(`index.html`, `manifest.json`, `service-worker.js`, `icons/`, `README.md`)을 저장소 루트에 업로드합니다
   - GitHub 웹사이트에서 "Add file → Upload files"로 드래그 앤 드롭 가능
3. 저장소의 **Settings → Pages** 메뉴로 이동
4. **Source**를 "Deploy from a branch"로 설정하고, 브랜치는 `main`, 폴더는 `/ (root)` 선택 후 저장
5. 1~2분 후 `https://내아이디.github.io/sydney-trip/` 주소로 접속 가능

---

## 3. 아이폰 홈 화면에 추가하기

1. 위에서 배포한 주소를 **아이폰 사파리(Safari)** 로 엽니다 (반드시 사파리여야 하며, 카카오톡 인앱 브라우저 등에서는 "홈 화면에 추가"가 보이지 않을 수 있습니다)
2. 하단(또는 상단)의 **공유 버튼**(□ 위에 화살표 아이콘)을 탭
3. 아래로 스크롤해서 **"홈 화면에 추가"** 선택
4. 이름을 확인하고 **추가** 탭
5. 홈 화면에 아이콘이 생기고, 탭하면 주소창 없이 전체 화면 앱처럼 실행됩니다
6. 이후에는 **인터넷이 없어도** 지금까지 저장된 일정을 보고 수정할 수 있어요 (서비스 워커가 앱 파일을 기기에 저장해두기 때문입니다)

---

## 4. 업데이트하는 방법

내용을 수정한 새 파일(또는 새 zip)이 생기면:

### Netlify Drop인 경우
- https://app.netlify.com/drop 에서 같은 사이트에 새 zip을 다시 드래그하면 자동으로 갱신됩니다 (무료 계정으로 사이트를 연결해둔 경우)

### GitHub Pages인 경우
- 저장소에 수정된 파일을 다시 업로드(커밋)하면 1~2분 내에 자동 반영됩니다

### 아이폰에 설치된 앱이 최신 내용을 못 받아올 때
- 서비스 워커는 앱 파일(HTML/CSS/JS/아이콘)을 오프라인용으로 저장해두기 때문에, 배포 후 최신 버전을 즉시 못 받아올 수 있습니다
- `service-worker.js` 파일 맨 위의 아래 부분을 열어서:
  ```js
  const CACHE_VERSION = 'sydney-trip-v1';
  ```
  숫자를 `v2`, `v3` 등으로 바꿔서 재배포하면, 다음에 앱을 열 때 새 버전을 받아오고 이전 캐시는 자동으로 정리됩니다
- 그래도 안 바뀌면: 홈 화면 앱을 완전히 종료(위로 스와이프해서 끄기) 후 다시 열어보세요

---

## 5. 여행 일정 데이터에 대해

- 일정 데이터(날짜별 장소, 시간, 메모, 체크리스트 완료 여부)는 **각자의 아이폰 브라우저 저장공간**에 저장됩니다
- 앱 안의 **"📤 친구에게 공유 (내보내기)"** 버튼으로 `.json` 파일을 만들어서 친구들에게 전달하고, 친구는 **"📥 받은 일정 불러오기"** 로 불러오면 동일한 일정을 볼 수 있어요
- 여행 시작일은 상단의 D-Day 배지를 탭해서 `YYYY-MM-DD` 형식으로 입력하면 D-Day/여행 중(Day N)/여행 완료로 자동 표시됩니다

즐거운 여행 되세요! 🌸
